# 02 — Number Triangle

← [Star Number Patterns hub](../README.md) · [One Pattern hub](../../README.md)

## What it prints

Centered rows: row `i` repeats digit `i` exactly `i` times.

## Code (same folder)

| Language | Location |
| -------- | -------- |
| Python | [`print_patterns.py`](../print_patterns.py) → `number_triangle` |
| Java | [`PrintPatterns.java`](../PrintPatterns.java) → `numberTriangle` |

## Run every pattern

```text
python print_patterns.py 5
java PrintPatterns 5
```

Change `5` to any size `n ≥ 1`. All loops use **1-based** outer indices (`1 … n`) in the source.
