# 04 — Number Increasing Reverse Pyramid

← [Star Number Patterns hub](../README.md) · [One Pattern hub](../../README.md)

## What it prints

Inverted: rows shorten from `1..n` down to `1`.

## Code (same folder)

| Language | Location |
| -------- | -------- |
| Python | [`print_patterns.py`](../print_patterns.py) → `number_increasing_reverse_pyramid` |
| Java | [`PrintPatterns.java`](../PrintPatterns.java) → `numberIncreasingReversePyramid` |

## Run every pattern

```text
python print_patterns.py 5
java PrintPatterns 5
```

Change `5` to any size `n ≥ 1`. All loops use **1-based** outer indices (`1 … n`) in the source.
