## What it prints

Filled `n × n` block of `*`.

| Language | Location |
| -------- | -------- |
| Python | [`print_patterns.py`](../print_patterns.py) → `square_fill` |
| Java | [`PrintPatterns.java`](../PrintPatterns.java) → `squareFill` |

## Run every pattern

```text
python print_patterns.py 5
java PrintPatterns 5
```

Change `5` to any size `n ≥ 1`. All loops use **1-based** outer indices (`1 … n`) in the source.
