# 06 — Zero–One Triangle

← [Star Number Patterns hub](../README.md) · [One Pattern hub](../../README.md)

## What it prints

Alternating `0`/`1`; row `i` starts with `1` when `i` is odd.

## Code (same folder)

| Language | Location |
| -------- | -------- |
| Python | [`print_patterns.py`](../print_patterns.py) → `zero_one_triangle` |
| Java | [`PrintPatterns.java`](../PrintPatterns.java) → `zeroOneTriangle` |

## Run every pattern

```text
python print_patterns.py 5
java PrintPatterns 5
```

Change `5` to any size `n ≥ 1`. All loops use **1-based** outer indices (`1 … n`) in the source.
