# 03 — Number Increasing Pyramid

← [Star Number Patterns hub](../README.md) · [One Pattern hub](../../README.md)

## What it prints

Left triangle: row `i` prints `1 2 … i`.

## Code (same folder)

| Language | Location |
| -------- | -------- |
| Python | [`print_patterns.py`](../print_patterns.py) → `number_increasing_pyramid` |
| Java | [`PrintPatterns.java`](../PrintPatterns.java) → `numberIncreasingPyramid` |

## Run every pattern

```text
python print_patterns.py 5
java PrintPatterns 5
```

Change `5` to any size `n ≥ 1`. All loops use **1-based** outer indices (`1 … n`) in the source.
