## What it prints

Solid centered diamond (same body as many “rhombus” sheets).

## Code (same folder)

| Language | Location |
| -------- | -------- |
| Python | [`print_patterns.py`](../print_patterns.py) → `rhombus_pattern` |
| Java | [`PrintPatterns.java`](../PrintPatterns.java) → `rhombusPattern` |

## Run every pattern

```text
python print_patterns.py 5
java PrintPatterns 5
```

Change `5` to any size `n ≥ 1`. All loops use **1-based** outer indices (`1 … n`) in the source.
