# 05 — Number Changing Pyramid (Floyd)

← [Star Number Patterns hub](../README.md) · [One Pattern hub](../../README.md)

## What it prints

Numbers increase continuously: `1`, then `2 3`, then `4 5 6`, …

## Code (same folder)

| Language | Location |
| -------- | -------- |
| Python | [`print_patterns.py`](../print_patterns.py) → `number_changing_pyramid` |
| Java | [`PrintPatterns.java`](../PrintPatterns.java) → `numberChangingPyramid` |

## Run every pattern

```text
python print_patterns.py 5
java PrintPatterns 5
```

Change `5` to any size `n ≥ 1`. All loops use **1-based** outer indices (`1 … n`) in the source.
