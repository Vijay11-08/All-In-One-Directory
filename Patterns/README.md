# Programming Patterns Collection

This directory contains a comprehensive collection of programming patterns (star patterns, number patterns, alphabet patterns) implemented in both **Python** and **Java**.

## Directory Structure

Each pattern is located in its own subfolder and includes:
- `pattern.png`: A high-quality visual representation of the pattern. (Generated for key patterns)
- `pattern.py`: Python implementation.
- `Pattern.java`: Java implementation.

### Number & Alphabet Patterns
10. **Pascal's Triangle**: Mathematical triangle of binomial coefficients.

12. **Number Right Half Pyramid**: Increasing numbers (1, 1-2, 1-2-3...).
13. **Inverted Number Right Half Pyramid**: Decreasing numbers.
14. **Number Pyramid**: Centered numbers.
